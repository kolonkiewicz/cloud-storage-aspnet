# Aplikacja chmurowa ASP.NET Core MVC

Webowa aplikacja umożliwiająca przechowywanie i zarządzanie plikami użytkowników.

## Technologie

- C#
- ASP.NET Core MVC
- Entity Framework
- SQLite

## Funkcjonalności

- rejestracja i logowanie użytkowników
- autoryzacja użytkowników
- przesyłanie plików
- zarządznie plikami
- dashboard dla administratora umożliwiający zarządzanie rolami kont użytkownika
