# Aplikacja chmurowa ASP.NET Core MVC

Webowa aplikacja chmurowa umożliwiająca bezpieczne przechowywanie i zarządzanie plikami użytkowników z poziomu przeglądarki.

## Technologie

- **Framework:** .NET 6.0 (ASP.NET Core MVC)
- **Język:** C#
- **Dostęp do danych:** Entity Framework Core
- **Baza danych:** SQLite
- **Frontend:** .cshtml (Razor), Bootstrap, CSS3

## Funkcjonalności

- **System kont użytkowników:** Rejestracja oraz logowanie (Authentication).
- **Autoryzacja użytkowników:** Zabezpieczenie dostępu do zasobów na podstawie zalogowanego profilu.
- **Zarządzanie plikami:** Przesyłanie (upload), pobieranie (download) oraz usuwanie plików przez użytkownika.
- **Panel administratora:** Dedykowany dashboard umożliwiający zaawansowane zarządzanie rolami kont użytkowników.
